# Инструкция по сборке Zapret UI

## Подготовка к сборке

### 1. Скачайте winws.exe

Zapret UI требует `winws.exe` для работы. Скачайте его из одного из источников:

**Вариант A: Оригинальный zapret**
```
https://github.com/bol-van/zapret/releases
```

**Вариант B: Сборка для Discord/YouTube**
```
https://github.com/Flowseal/zapret-discord-youtube/releases
```

### 2. Создайте структуру папок

```
zapret-ui/
├── bin/
│   └── winws.exe          ← Поместите сюда
├── lists/
│   └── hostlist.txt       ← Создайте или скопируйте
├── strategies/            ← Уже есть в репозитории
└── assets/                ← Уже есть в репозитории
```

### 3. Создайте hostlist.txt

Минимальный пример `lists/hostlist.txt`:

```
discord.com
youtube.com
googlevideo.com
ytimg.com
```

Или скопируйте готовый из репозитория zapret.

## Сборка на Windows

### Установка зависимостей

```bash
pip install -r requirements.txt
```

### Запуск сборки

```bash
# Вариант 1: Через .bat (рекомендуется)
scripts\build_windows.bat

# Вариант 2: Напрямую через Python
python scripts\build.py
```

### Результат

После успешной сборки:
```
dist/
└── ZapretUI.exe    ← Готовый исполняемый файл
```

## Устранение проблем

### Ошибка: "Unable to find 'bin'"

**Причина**: Отсутствует папка `bin/` или `winws.exe`

**Решение**: Скачайте `winws.exe` и поместите в `bin/winws.exe`

### Ошибка: "Unable to find 'lists'"

**Причина**: Отсутствует папка `lists/` или `hostlist.txt`

**Решение**: Создайте `lists/hostlist.txt` с доменами

### Предупреждение: "папка X не найдена, пропускаем"

**Это нормально** — сборка продолжится без этой папки. Но для полной функциональности нужны:
- `bin/winws.exe` — обязательно
- `lists/hostlist.txt` — обязательно
- `strategies/` — уже есть в репозитории
- `assets/` — уже есть в репозитории

---

## Проверка сборки

После сборки проверьте:

1. **Размер файла**: ~50-80 МБ (нормально)
2. **Запуск**: Двойной клик на `ZapretUI.exe`
3. **Права администратора**: Запустите от имени администратора

---

## Распространение

Для распространения нужно включить:

```
ZapretUI.exe          ← Основной файл
bin/winws.exe         ← Если не встроен в .exe
lists/hostlist.txt    ← Если не встроен в .exe
```

**Примечание**: При сборке с `--onefile` все файлы упаковываются в один `.exe`, но `winws.exe` и `hostlist.txt` извлекаются во временную папку при запуске.

## Альтернативная сборка (onedir)

Для сборки в папку вместо одного файла:

Измените в `scripts/build.py`:
```python
"--onefile",  # Удалите эту строку
"--onedir",   # Добавьте эту строку
```

Результат:
```
dist/
└── ZapretUI/
    ├── ZapretUI.exe
    ├── _internal/
    └── ... (библиотеки)
```

**Преимущества onedir**:
- Быстрее запускается
- Меньше проблем с антивирусами
- Проще отладка

**Недостатки**:
- Больше файлов для распространения
