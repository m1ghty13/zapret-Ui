# Zapret UI v2.1

Графический интерфейс для [zapret](https://github.com/bol-van/zapret) с системным треем.

## ✨ Особенности

- 🪟 **Нативное окно** (PyWebView)
- 🎯 **Системный трей** с управлением
- ⚡ **Работа в фоне** при закрытом окне
- 🔄 **Смена стратегии** из трея
- 📦 **Сборка в .exe** для Windows
- 🎨 **Современный React UI**
- 🔔 **Уведомления**

## 🚀 Быстрый старт

### Для пользователей Windows

1. Скачайте `ZapretUI.exe`
2. Запустите от имени администратора
3. Готово!

### Для разработчиков

```bash
# Запустить (зависимости установятся автоматически)
python main.py
```

**Примечание**: Node.js НЕ требуется! React уже собран в dist/

### Сборка .exe

```cmd
build_windows.bat
```

Подробнее: [BUILD_WINDOWS.md](BUILD_WINDOWS.md)

## 📋 Использование

### Основное окно

1. Выберите домены (Discord, YouTube, и т.д.)
2. Выберите стратегию (ALT, ALT2, ALT3...)
3. Нажмите "Запустить"
4. Закройте окно → приложение в трее

### Системный трей

**ПКМ по иконке:**

```
┌─────────────────────────────┐
│ Zapret UI                   │
├─────────────────────────────┤
│ Статус: ✓ Активен (ALT3)   │
├─────────────────────────────┤
│ Стратегии ►                 │
│   ✓ ALT                    │
│     ALT2                    │
│     ALT3                    │
├─────────────────────────────┤
│ Остановить                  │
│ Открыть UI                  │
│ Выход                       │
└─────────────────────────────┘
```

**Возможности:**
- ✅ Запуск/остановка защиты
- ✅ Смена стратегии (первые 10)
- ✅ Просмотр статуса
- ✅ Открытие UI
- ✅ Полное закрытие

## 🎯 Типичные сценарии

### Сценарий 1: Настроить и забыть
```
1. Запустить Zapret UI
2. Выбрать домены
3. Запустить защиту
4. Закрыть окно
5. Забыть - всё работает в фоне
```

### Сценарий 2: Быстрая смена стратегии
```
1. ПКМ по иконке в трее
2. Стратегии → ALT5
3. Защита перезапускается автоматически
```

### Сценарий 3: Минимальное потребление
```
1. Настроить и запустить
2. Закрыть окно
3. RAM: 80 MB вместо 125 MB
```

## 📊 Потребление ресурсов

| Режим | RAM | CPU |
|-------|-----|-----|
| С открытым окном | ~125 MB | ~0.5% |
| Только в трее | ~80 MB | ~0.3% |
| Только winws.exe | ~15 MB | ~0.1% |

**Вывод**: Закрытое окно экономит 45 MB RAM!

## 🔧 Настройки

### Settings → Minimize to tray

- **ON** (по умолчанию): Закрытие окна = сворачивание в трей
- **OFF**: Закрытие окна = полное закрытие

## ❓ FAQ

### Q: Нужны ли права администратора?
**A**: Да, для запуска winws.exe

### Q: Можно ли закрыть окно?
**A**: Да! Приложение продолжит работать в трее

### Q: Как сменить стратегию без UI?
**A**: ПКМ по иконке в трее → Стратегии

### Q: Где хранятся настройки?
**A**: `%APPDATA%/ZapretUI/config.json`

### Q: Как полностью закрыть приложение?
**A**: ПКМ по иконке в трее → Выход

## 🐛 Известные проблемы

### Windows: Иконка в скрытых значках
**Решение**: Перетащите иконку на панель задач

### Антивирус блокирует .exe
**Решение**: Добавьте в исключения или соберите сами
