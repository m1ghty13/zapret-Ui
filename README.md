## ✨ Возможности

- 🖥️ **Нативный интерфейс** — PyQt6 без браузерного движка
- 🎯 **Системный трей** — управление без открытого окна
- ⚡ **Легковесность** — ~50-80 МБ RAM (вместо ~150-200 МБ)
- 🔄 **Quick Switcher** — быстрое переключение последних 5 стратегий
- 📦 **Экспорт в .bat** — создание standalone bundle для запуска без GUI
- 🎨 **Темы и акценты** — тёмная/светлая/системная тема + выбор цвета
- 🌐 **Мониторинг подключения** — индикатор статуса интернета в трее
- 🔁 **Auto-recovery** — автоматический перезапуск при падении winws.exe
- 🔔 **Уведомления** — о статусе работы и событиях
- 📊 **Авто-тест стратегий** — автоматический подбор лучшей стратегии

## 🚀 Быстрый старт

### Для пользователей Windows

1. **Скачайте** `ZapretUI.exe` из [Releases](https://github.com/m1ghtyqxddd/zapret-ui/releases)
2. **Запустите** от имени администратора
3. **Готово!** Выберите домены и стратегию

### Для разработчиков

```bash
# Клонировать репозиторий
git clone https://github.com/m1ghtyqxddd/zapret-ui.git
cd zapret-ui

# Установить зависимости
pip install -r requirements.txt

# Запустить
python main.py
```

### Сборка .exe

```bash
# На Windows
python scripts/build.py

# Результат: dist/ZapretUI.exe
```

## 🎯 Типичные сценарии

### Сценарий 1: Настроить и забыть
```
1. Запустить Zapret UI
2. Выбрать домены (Discord, YouTube...)
3. Запустить защиту
4. Закрыть окно → работает в трее
```

### Сценарий 2: Быстрая смена стратегии
```
1. ПКМ по иконке в трее
2. Стратегии → выбрать другую
3. Защита перезапускается автоматически
```

### Сценарий 3: Автоматический подбор
```
1. Панель "Стратегии"
2. Нажать "Авто-тест"
3. Лучшая стратегия применится сама
```

## 📊 Производительность

| Режим | RAM | CPU | Описание |
|-------|-----|-----|----------|
| **Окно открыто** | ~80 МБ | ~0.5% | Полный интерфейс |
| **Только трей** | ~50 МБ | ~0.3% | Фоновый режим |

- 🚀 Мгновенный запуск
- 🔋 Минимальная нагрузка на CPU

## 🔧 Настройки

### Основные параметры

| Параметр | Описание | По умолчанию |
|----------|----------|--------------|
| **Minimize to tray** | Сворачивание в трей при закрытии | ✅ Включено |
| **Autostart** | Запуск с Windows | ❌ Выключено |
| **Ping Monitor** | Индикатор подключения в трее | ❌ Выключено |
| **Auto-recovery** | Автоперезапуск при падении | ❌ Выключено |
| **Theme** | Тёмная/Светлая/Системная | Тёмная |
| **Accent Color** | Цвет акцента интерфейса | Синий (#007aff) |
| **Strategy** | Текущая стратегия обхода | ALT |
| **Hostlist** | Путь к списку доменов | `lists/hostlist.txt` |

### Новые возможности v2.2

#### 🎨 Темы и акценты
- Выбор темы: тёмная, светлая или системная
- Кастомный цвет акцента с пресетами
- Мгновенное применение без перезапуска

#### 📦 Экспорт стратегий
- Кнопка 📦 на каждой стратегии
- Экспорт только .bat файла
- Полный bundle с winws.exe, hostlist.txt, start.bat, stop.bat

#### 🌐 Мониторинг подключения
- TCP проверка к 1.1.1.1:443 каждые 5 сек
- Индикатор на иконке трея (зелёный/красный)
- Toast-уведомления при потере/восстановлении

#### 🔁 Auto-recovery
- Мониторинг процесса winws.exe каждые 3 сек
- Автоматический перезапуск при падении
- Уведомление о срабатывании recovery

## ❓ FAQ

<details>
<summary><b>Нужны ли права администратора?</b></summary>

Да, для запуска `winws.exe` требуются права администратора.
</details>

<details>
<summary><b>Можно ли закрыть окно?</b></summary>

Да! По умолчанию закрытие окна сворачивает приложение в трей. Защита продолжает работать.
</details>

<details>
<summary><b>Как сменить стратегию без открытия окна?</b></summary>

ПКМ по иконке в трее → Стратегии → выбрать нужную.
</details>

<details>
<summary><b>Где хранятся настройки?</b></summary>

`%APPDATA%/ZapretUI/config.json`
</details>

<details>
<summary><b>Как полностью закрыть приложение?</b></summary>

ПКМ по иконке в трее → Выход
</details>

<details>
<summary><b>Антивирус блокирует .exe</b></summary>

Добавьте в исключения или соберите .exe самостоятельно из исходников.
</details>

<details>
<summary><b>Иконка не видна в трее</b></summary>

Windows может скрывать иконку. Перетащите её из скрытых значков на панель задач.
</details>
